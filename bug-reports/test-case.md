## 📝 Как запустить тесты

### Ручное тестирование:
1. Открыть приложение
2. Выполнить шаги из каждого тест-кейса
3. Сравнить результат с ожидаемым

### Автоматизированное тестирование:
```bash
# Запуск всех unit-тестов
./gradlew test

# Запуск конкретного теста
./gradlew test --tests "com.apex.karting.data.ApexRepositoryTest.testIndividualSmsTimer"
