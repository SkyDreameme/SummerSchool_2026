```mermaid
flowchart TD
    Entry((Старт приложения)) --> SCR-001

    subgraph TabBar [Нижняя навигация (Tab Bar)]
        direction LR
        SCR-002[SCR-002: Расписание слотов]
        SCR-005[SCR-005: Мои записи]
        SCR-007[SCR-007: Профиль]
    end
    
    SCR-002 -->|Тап по слоту| SCR-003[SCR-003: Детали слота]
    SCR-003 -->|Кнопка 'Забронировать'| SCR-004[SCR-004: Бронирование]
    SCR-004 -->|Переход по depositLink| External((Внешний банк / СБП))
    
    SCR-005 -->|Тап по записи| SCR-006[SCR-006: Детали брони]
    SCR-006 -->|Кнопка 'Отменить'| SCR-009[SCR-009: Подтверждение отмены]
    SCR-006 -->|Push / Кнопка 'Оценить'| SCR-008[SCR-008: Оценка маршала]
    
    SCR-009 -->|Подтвердить| SCR-005
    SCR-008 -->|Отправить| SCR-005
```    

### Легенда переходов

- **Сплошная стрелка** — прямой пользовательский переход (тап, CTA).
- **Пунктирная стрелка** — переход по push-уведомлению (информационному, без action-кнопок согласно FR-13).
- **Внешний узел `External`** — выход из приложения в банк/СБП по `depositLink` (NFR-5: эквайринг не встраивается).

## 3. Границы вне скоупа (Out of Scope)

Согласно NFR и ограничениям текущей поставки, в приложении **явно НЕ реализуются**:

1. **Визуальная карта маршрута** — отображается только текстовое описание конфигурации трассы (Short / Long). *(NFR-8)*
2. **Онлайн-оплата внутри приложения** — эквайринг, банковские карты и СБП не встраиваются; используется только переход по внешней ссылке `depositLink` на перевод по номеру телефона. *(NFR-5)*
3. **Групповые брони и лист ожидания** — поддерживается только сценарий «1 клиент = 1 бронь». *(NFR-6)*
4. **Waivers и цифровые согласия** — загрузка документов, возрастные и медицинские ограничения не реализуются. *(NFR-7)*
5. **Валидация «опытности» клиента** — приложение не проверяет навыки пользователя при выборе конфигурации трассы. *(NFR-8)*
6. **Автоматические компенсации и возвраты** — в том числе при отмене заезда по инициативе центра. *(NFR-9)*
7. **Интерфейсы ролей «Маршал» и «Администратор»** — приложение предназначено исключительно для клиентов. *(NFR-10)*
8. **Публичный рейтинг маршала** — оценки собираются только для внутренней аналитики, в карточках слотов не отображаются. *(FR-18)*
9. **Текстовые комментарии к оценке** — форма оценки маршала содержит только выбор от 1 до 5 звёзд. *(FR-14)*
10. **Action-кнопки в Push-уведомлениях** — все push (напоминания, запрос оценки, отмена центром) являются строго информационными. *(FR-13)*
11. **Клиентская атомарная проверка свободных мест** — ответственность бэкенда; приложение только корректно обрабатывает ответ «мест нет». *(NFR-1, NFR-2)*
