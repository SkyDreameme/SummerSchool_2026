# BS-003: Фильтр дат расписания

## Пользовательская цель
Расширить горизонт просмотра слотов за пределы стандартных 7 дней и выбрать конкретную дату.

## Контекст и триггер открытия
- **Родительский экран:** SCR-002 «Расписание слотов».
- **Триггер:** тап по элементу «Расширить диапазон» / иконке календаря в горизонтальном переключателе дат.
- **Тип шторки:** функциональный фильтр (functional filter).

## Контент и данные на шторке
- Заголовок: «Выберите дату».
- Встроенный календарь (месяц / два месяца).
- Подсветка дат, на которые есть хотя бы один слот (если бэкенд отдаёт такой признак).
- Серые / недоступные даты — за пределами рабочего расписания центра.
- Кнопки действий:
    - **Основная:** «Применить» — закрывает шторку и перезагружает список слотов на SCR-002 с новым диапазоном.
    - **Вторичная:** «Сбросить» — возвращает горизонт к стандартным 7 дням.
- Состояние загрузки после применения фильтра.

## Связанные требования

### User Stories
- **US-2** — Как клиент, я хочу видеть расписание свободных слотов на 7 дней вперёд, чтобы выбрать удобное время заезда.

### Use Cases
- **UC-2** — Просмотр расписания: клиент открывает экран, видит слоты на 7 дней, при необходимости расширяет диапазон фильтром дат.

### Functional Requirements
- **FR-2** — Система должна отображать расписание слотов на 7 дней вперёд по умолчанию, с возможностью расширения горизонта через фильтр дат.

### Non-Functional Requirements
- **NFR-1** — Бэкенд является источником истины; если на выбранную дату слотов нет — SCR-002 показывает empty state (BS-003 не блокируется).
