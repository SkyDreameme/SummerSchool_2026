3. Назначение и контекст
   3.1. Краткое описание
   Модалка/экран оценки маршала: 5 звёзд, без текстового комментария. Открывается по push через 1 час после заезда или из SCR-006.

3.2. Пользовательская цель
Клиент хочет быстро оценить работу маршала, не заполняя сложных форм.

3.3. Бизнес-ценность
Обратная связь для улучшения сервиса (внутренняя аналитика).

3.4. Контекст открытия
Сессия: авторизован
Сеть: обязательна
Предшествующее действие: push-уведомление через 1 час после окончания заезда / тап по CTA «Оценить маршала» в SCR-006

4. Навигация
   4.1. Входящая навигация
   | Источник | Триггер | Условие |
   | --- | --- | --- |
   | Push-уведомление | Push через 1 час после заезда | payload.slotId, проверка rated == false |
   | SCR-006 | CTA «Оценить маршала» | status == Completed и rated == false и now ≥ endsAt |

4.2. Исходящая навигация
| Действие | Целевой экран | Тип перехода |
| --- | --- | --- |
| Успех (201) | BS-006 | bottom sheet |
| BS-006 → «Закрыть» | SCR-005 / закрыть модалку | pop / dismiss |
| Ошибка 409 ALREADY_RATED | Закрыть с тостом «Вы уже оценили этот заезд» | dismiss |
| Ошибка 422 RATING_NOT_ALLOWED | Закрыть с тостом «Оценка недоступна» | dismiss |

4.3. Системные жесты
Swipe-down: разрешён (закрытие модалки).

5. Данные
   5.1. Локальные модели
   | Сущность | Атрибуты | RW/RO | Источник |
   | --- | --- | --- | --- |
   | RatingDraft | slotId, rating (1-5) | RW | Локально |

5.2. API-эндпоинты
| Метод | Endpoint | Назначение | Обрабатываемые коды |
| --- | --- | --- | --- |
| POST | api/instructors/api.yaml → /slots/{slotId}/ratings | Отправка оценки | 201, 404, 409 (ALREADY_RATED), 422 (RATING_NOT_ALLOWED) |

5.3. Кэширование и актуальность
Источник истины: бэкенд.

6. Структура экрана
   6.1. Блок: Контекст заезда
   Назначение: дата и время заезда
   Компоненты: `Date`, `Time`
   Данные: из payload push / из SCR-006

6.2. Блок: Шкала оценки
Назначение: выбор 1–5 звёзд
Компоненты: `StarRating` (5 звёзд, тап фиксирует значение)
Данные: RatingDraft.rating

6.3. CTA
| CTA | Тип | Активен при | Действие |
| --- | --- | --- | --- |
| «Отправить оценку» | primary | rating выбрано (1-5) | POST /slots/{slotId}/ratings |

7. Состояния экрана
   | Состояние | Триггер | Визуализация |
   | --- | --- | --- |
   | Loading (отправка) | Тап по CTA | Inline-спиннер на кнопке |
   | Error (inline) | 409 / 422 | Тост с текстом ошибки |
   | Success | 201 | Открытие BS-006 |

7.1. Текст состояний
| Состояние | Заголовок | Подсказка |
| --- | --- | --- |
| 409 ALREADY_RATED | — | «Вы уже оценили этот заезд» |
| 422 RATING_NOT_ALLOWED | — | «Оценка доступна только после завершения заезда» |
| Сеть | — | «Нет соединения. Проверьте интернет» |

8. Логика и взаимодействия
   8.1. Инициализация
1. Получить slotId из payload push / из SCR-006.
2. Проверить локально: если rated == true → показать тост «Вы уже оценили этот заезд» и закрыть (п. 11 из review).
3. Отрисовать форму.

8.2. Happy path
1. Пользователь получает push / тапает CTA в SCR-006.
2. Открывается модалка.
3. Пользователь выбирает 1–5 звёзд.
4. Тапает «Отправить оценку».
5. Приложение → POST /slots/{slotId}/ratings → 201.
6. UI открывает BS-006.

8.3. Бизнес-правила
| # | Правило | Обоснование |
| --- | --- | --- |
| 1 | Одна оценка на слот от клиента | data-model §2.5 |
| 2 | Текстовый комментарий не предусмотрен | FR-14 |
| 3 | Push не отправляется, если заезд отменён центром | П. 12 из review |
| 4 | Оценка доступна только после завершения заезда | П. 13 из review |

11. Критерии приёмки (DoD)
- [ ] Выбор 1-5 звёзд работает.
- [ ] Отправка оценки → BS-006.
- [ ] Обработка 409 / 422.
- [ ] Нет поля для текста.

12. Edge cases и ограничения
    | # | Сценарий | Поведение |
    | --- | --- | --- |
    | 1 | Push-дубликат (клиент уже оценил) | Проверка rated → тост «Вы уже оценили» + закрытие (п. 11 из review) |
    | 2 | Заезд отменён центром | Push не приходит (п. 12 из review); если открыли из SCR-006 — CTA скрыта |
    | 3 | Открыли из SCR-006 до завершения заезда | CTA «Оценить маршала» скрыта (п. 13 из review) |
    | 4 | 409 от бэкенда | Тост «Вы уже оценили этот заезд» + закрытие |
    | 5 | 422 от бэкенда | Тост «Оценка доступна только после завершения заезда» + закрытие |

12.1. Явные ограничения скоупа
- Текстовые комментарии — вне скоупа (FR-14).
- Публичный рейтинг — вне скоупа (FR-18).
