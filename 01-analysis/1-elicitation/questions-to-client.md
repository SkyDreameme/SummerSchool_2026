## Итоговое резюме

По результатам уточнений зафиксированы следующие ключевые решения, которые вошли в `domain-description.md`:

1. **Скоуп роли:** только клиент. Маршал и админ — вне текущей поставки.
2. **Авторизация:** единственная — по SMS на номер телефона.
3. **Бронирование:** 1 клиент = 1 бронь, без групповых заявок, без валидации опыта, без waivers.
4. **Оплата:** через ссылку на перевод по номеру телефона (без встроенного эквайринга).
5. **Отмены:** жёсткое правило «2 часа»; лист ожидания не делается.
6. **Оценки маршалам:** только 1–5 звёзд, без текста, без публичного рейтинга.
7. **Уведомления:** 3 типа push (24ч / 2ч до старта, 1ч после окончания). Без action-кнопок.
8. **Лояльность:** 3 статуса (нет / бронза 10% / серебро 20%), определяется бэкендом.
9. **История:** раздел «Мои заезды» обязателен.
